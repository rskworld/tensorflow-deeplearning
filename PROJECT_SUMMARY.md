## Quick Start

1. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Run examples**:
   ```bash
   python main.py --module neural_networks
   ```

3. **Start API server**:
   ```bash
   python api/server.py
   ```

4. **Run with Docker**:
   ```bash
   docker-compose up
   ```
